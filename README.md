
  # AutiCare 
  AutiCare is a web application designed to connect parents of autistic children with qualified professionals such as therapists, educators, and support workers. The platform allows parents to find and book appointments with professionals, while also providing resources and support for managing their child's care.
  
  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
